━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Это автоматическая запись — не нужен OBS/Loom/CapCut.
Скрипт сам открывает headless Chromium 1920×1080,
прокликивает 8 слайдов с правильным таймингом и сохраняет
`slides.webm` в video-build/f2/.

КОМАНДЫ (Git Bash или PowerShell, из корня проекта):

  cd C:/Projects/custos/video-build/f2
  npm install                 # один раз (Playwright Chromium ~150 МБ)
  node record-slides.mjs

ОЖИДАНИЕ: ~2 минуты (109 сек запись + старт/закрытие браузера).

РЕЗУЛЬТАТ: video-build/f2/slides.webm
  - 1920×1080, ~109 сек, БЕЗ звука
  - 8 слайдов с тайминговыми блоками:
      s1 16s | s2 16s | s3 14s | s4 13s
      s5 14s | s6 14s | s7 12s | s8 10s

ПРОВЕРКА: открой slides.webm в плеере (VLC / MPC / Windows Media).
Все count-up анимации должны проиграться, переходы между
слайдами должны быть видны (нажатия ArrowRight скрипт делает сам).

ЕСЛИ slides.webm битый/пустой:
  - Проверь что deck-v2.html открывается в обычном Chrome
    (file:// путь): C:\Projects\custos\assets\pitch-slides\deck-v2.html
  - Перезапусти `npm install` в video-build/f2/
  - Запусти `node record-slides.mjs` повторно
