## Reproducing

Requires a funded devnet keypair at
`~/.config/solana/id.json` (run `npm run setup:devnet` to create one
and top it up from the faucet).

```bash
npm run smoke:create                                              # prints MULTISIG_PDA
npm run smoke:nonce-plan                                          # prints NONCE_PUBKEY, writes .smoke-nonce.json
# Populate .env: CUSTOS_RPC_URL, CUSTOS_WS_URL, CUSTOS_CLUSTER=devnet,
#               CUSTOS_WATCH=<SQUADS_PROGRAM>:<MULTISIG_PDA>,11111111111111111111111111111111:<NONCE_PUBKEY>
npm run dev                                                       # in a second terminal
npm run smoke:timelock -- <MULTISIG_PDA>                          # expect CRITICAL
npm run smoke:weaken   -- <MULTISIG_PDA>                          # expect HIGH
npm run smoke:nonce-init                                          # expect CRITICAL
```
